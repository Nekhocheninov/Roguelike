## How To Start

1. Clone the repo:
    ```
    git clone https://github.com/Nekhocheninov/Tiny-Roguelike.git
    ```

2. Open index.html.

[Or click here go to the website.](https://nekhocheninov.github.io/Tiny-Roguelike/)

## Controls
- WASD - Move;
- Space - Attack;
- R - Use potion from inventory.

Have a good game!
